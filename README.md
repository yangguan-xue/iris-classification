Iris数据集决策树分类项目

项目概述

一个基于决策树算法的Iris鸢尾花分类器，在测试集上达到97.78%的准确率。项目包含完整的机器学习流水线：数据加载、模型训练、评估和可视化。

📊 实验结果
准确率: 97.78% (44/45个测试样本正确)
训练时间: 0.001秒
特征重要性:
花瓣长度: 55.09%
花瓣宽度: 44.91%
花萼特征: 0.00%
关键发现: 花瓣特征是区分鸢尾花品种的主要依据

🧠 算法详情
算法: 决策树分类器
最大深度: 3层（防止过拟合）
评估指标: 准确率、精确率、召回率、F1分数
数据集划分: 70%训练集, 30%测试集
